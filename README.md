# Splatoon Rivals Pack

A Fleasion rivals texture pack based on Splatoon :3

## Resources

This pack includes:
- **Logo**: Splatoon X Battle icon
- **Textures**: Custom Splatoon-themed textures
- **Fonts**: Custom fonts for the pack

## Installation

Add the raw GitHub URLs to your Fleasion config to use these resources directly.

### Resource URLs

- Logo: `https://raw.githubusercontent.com/zky0ta/Splatoon-Rivals-Pack/main/resources/logo/S3_Icon_X_Battle.svg`
- Fonts folder: `https://raw.githubusercontent.com/zky0ta/Splatoon-Rivals-Pack/main/resources/fonts/`
